### 📼 Appearance
Text의 모든 단어들을 Parsing하여 개수를 세고 선택한 정렬방식에 따라 정렬한다.


 ### 📃 정렬 기능
 ![정렬 기능](https://im3.ezgif.com/tmp/ezgif-3-c90b29cd4183.gif)
 
 ./{실행파일명} {텍스트} {정렬방식}
 ex) ./WordCnt data.txt A
 
 정렬방식 : A(Ascending), D(Descending), M(Max), I(Min)
